Index

Ways of declare CSS styles

  1. Inline

    Плюсы

    • Можно быстро прописывать стиль для определенного элемента

    Минусы

    • Необходимо прописывать стили для каждого элемента
    • Сложно редактировать

    Можно использовать для почтовых клиентов. Они не все поддерживают добавление внешних стилей

  2. Head

    Плюсы

    • Можно прописывать стили для нескольких элементов

    Минусы

    • Необходимо добавлять стили для каждой страницы

    Можно использовать для одностраничных макетов

  3. Outter file

    Плюсы

    • Можно применить стили ко всему сайту
    • Удобно редактировать
    • Файл кэшируется

    Большинство ситуаций, которые поддерживают подключение внешних файлов
