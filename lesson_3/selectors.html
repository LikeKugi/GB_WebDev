Selector

Селектор - элемент HTML, который необходимо изменить

Виды селекторов

  1. Тэги

    в CSS файле выбирается просто тэг

                    /* Выбрать все тэги h1 */
                h1 {
                    color: blue;
                }

  2. ID

    Выбрать элемент с определённым ID

                    /* Выбрать тэг с определённым ID */
                #first {
                    background: #ccc;
                }

    ID - уникальный эоемент на странице. Подходит для стилизации конкретного блока. Не рекомендуется для стилизации

  3. Class

    Выбрать элементы с определенным классом

                    /* Выбрать все тэги на странице с определённым классом */
                .border {
                    border: 1px solid black;
                }

    Классы могут совпадать. У одного элемента может быть несколько классов. Более гибкие возможности для стилизации
