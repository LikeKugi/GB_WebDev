таблицы

генератор таблиц

плюсы

  • таблицы не перекрываются друг с другом при маленьких разрешениях
  • легко сделать кроссбраузерный дизайн
  • создание табличных элементов

Минусы

  • очень много лишнего кода
  • громоздкий код - не каждый дизайн можно создать с помощью таблиц
  • совсем непопулярная
  • долгая загрузка

атрибуты для объединения табличных ячеек

  • rowspan - объединение строк (по вертикали)
  • colspan - объединение столбцов (по горизонтали)
