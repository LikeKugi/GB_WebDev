Базовые свойства стилей

  1. Единицы измерения

    Относительные

    Предпочтительно применяются. Размеры относительно экрана

    • px - пиксель
    • % - процент
    • em - Высота текущего шрифта (16px - по стандарту)
    • rem - Относительно главного шрифта
    • ...

    Абсолютные

    Редко применяются. Не адаптивны

    • cm - сантиметр
    • mm - миллиметр
    • in - дюйм
    • pt - пункт
    • ...

  2. Единицы измерения цвета

    • Именованные

    • Функциональные RGBA

    • Шестнадцатиричные RGB

  3. Параметры ширины и высоты

    Редко задаются к текстовым элементам

    • Ширина

                              h1 {
                            width: 300px;
                        }

    • Высота

                              h1 {
                            height: 200 px;
                        }

  4. Фон элемента

    • Цвет фона

                              background-color: #ff0;

    • Изображение

                              background-image: url(path)

    • Повтор картинки

                              background-repeat: no-repeat;
      • repeat-x
      • repeat-y
      • no-repeat

    • Позиция заднего фона

                              background-position: top;
      • top
      • bottom
      • left
      • right

    • Фиксированный фон

                              background-attachment: fixed

    • Размер

                              background-size: cover;
      • cover - растянуть
      • contain - чтобы поместить полностью по ширине или по высоте
      • 100% - в процентах

  5. Добавление рамки

    • Цвет

                              border-color: red;

    • Стиль

                              border-style: solid;
      • solid
      • dotted
      • dashed
      • groove
      • ridge
      • double
      • inset
      • outset

    • Ширина

                              border-width: 2px;
